# CALF plotting

This directory contains the reproducible plotting workflow for the
article figures generated from experiment CSV snapshots.

## Main command

From the source-code repository root:

```bash
cd scl-plotting
uv run python -m calf_plotting --output-dir ../gfx
```

By default, the TD3-family figures use:

```text
expdata/cleared-enriched
```

You can override this root when needed:

```bash
uv run python -m calf_plotting \
  --data-root expdata/cleared-enriched \
  --output-dir /tmp/calf-figures
```

## Data roots

- TD3, Residual TD3, and Ours with TD3 backbone:
  `expdata/cleared-enriched`
- SAC, Residual SAC, and Ours with SAC backbone:
  `expdata/sac/cleared_td3params_preview`
- Legacy SAC schedule sensitivity curve:
  `expdata/sac/cleared/robot/sac_calfq.csv`
- TD3 drone ablations:
  `expdata/ablation/calfq_td3_drone`
- Checkpoint lambda-distance plot:
  `expdata/checkpoint_lambda_distance`

The bundle contains the CSV snapshots read by the default reproduction
commands; unused intermediate snapshots are intentionally not included.

## Generate a single figure

Use this pattern when only one figure is needed:

```bash
uv run python - <<'PY'
from pathlib import Path
from calf_plotting.paper_figures import generate_episode_return_comparison

generate_episode_return_comparison(Path("../gfx"))
PY
```

Replace `generate_episode_return_comparison` with the generator from the
table above.

The older wrapper scripts in this directory are kept for convenience for
some figures, but `python -m calf_plotting` and the generator functions
above are the canonical interface.

## Reproduce article tables

The final metrics tables in `main.tex` are generated by:

```bash
uv run python generate_final_metrics_tables.py \
  --output-dir /tmp/calf-tables
```

This writes:

```text
/tmp/calf-tables/final_metrics_tables.tex
```

The generator uses the same data roots as the figures:

- TD3-family rows from `expdata/cleared-enriched`
- SAC-family rows from `expdata/sac/cleared_td3params_preview`
- Robot baseline row from
  `expdata/cleared-enriched/robot/robot_nominal_eval_summary.csv`

The drone baseline row is kept as the reported nominal-policy constant
from the article snapshot: `100 $\pm$ 0` goal-reaching and
`0.54 $\pm$ 0.18` avoidance score.

## Non-plotting figures

The article also uses standalone LaTeX/TikZ figures in `gfx/build/`.
They are documented in `gfx/README.md` because they are not generated by
this plotting package.
